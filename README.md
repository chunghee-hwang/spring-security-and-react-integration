## 실행 방법

### 1. 백엔드 실행
[Visual Studio Code 실행](https://sambalim.tistory.com/67) 기준<br />
ServletInitializer.java 로 이동 후, F5 버튼 누르기

### 2. 프론트엔드 실행
```shell
cd frontend
npm install
npm start
```

### 로그인 화면 접속
1. http://localhost:3000 이동
2. 아이디(hch0821), 비밀번호(1234)를 입력하면 접근 가능한 페이지로 이동됨
